# SAFI-Blog
My blog is a web-based application using my own api that i created while practicing my learning of the api's world
## Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yahya03safi/My-first-API.git
2. **Navigate to the project directory**:
   ```bash
   cd My-first_API
3. **install npm files**:
   ```bash
   npm install
4. **Run on two differnt terminals the solution.js file and the server.js file**:
   ```bash
   nodemon solution.js
   nodemon server.js
5. **In your browser type**:
   ```bash
   localhost:3000
